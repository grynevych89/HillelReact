# React Homework

## Project Description
Educational project demonstrating the creation and usage of React Button and Input components with props and event handlers.

## Technologies
- React
- Vite
- JavaScript/JSX

## Installation and Setup

1. Clone the repository:
```bash
git clone [your-repo-url]
```

2. Install dependencies:
```bash
npm install
```

3. Run the project:
```bash
npm run dev
```

## Features
- Custom Button component with props (text, type, onClick)
- Custom Input component with props (placeholder, type, onChange)
- Interactive form with state management
- Clean and responsive UI design

## Components

### Button Component
- Accepts `text`, `type`, and `onClick` props
- Customizable styling
- Hover effects

### Input Component
- Accepts `placeholder`, `type`, `onChange`, and `value` props
- Controlled component
- Focus states
